Part 1 — Design & Implementation

This is a single-page web app that uses HTML, CSS and JavaScript (with jQuery) and the TMDb API.

  • Home/Search: live search bar with debounce and filters (genre, rating).
  • Click a movie card to view detailed info (full overview, genres, release date, rating).
  • Watchlist stored in localStorage; you can add/remove and mark watched/unwatched.
  • Smooth animations implemented using jQuery (.fadeIn/.fadeOut/.slideDown)

Part 2 — Key features

  • Search by title (live results)
  • Filter results by genre and minimum rating
  • Clickable movie cards show movie details
  • Add/remove movies to watchlist and mark as watched
